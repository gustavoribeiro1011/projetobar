# **Sobre**
Projeto Bar surgiu em 28 de Outubro de 2019 com a idéia de se tornar um sistema de gerenciamento de bares e restaurantes. 

### Funcionalidades:
- Gerenciamento de mesas
- controle de produtos,
- cardápio online,
- reservas de mesas
- monitoramento de pedidos
